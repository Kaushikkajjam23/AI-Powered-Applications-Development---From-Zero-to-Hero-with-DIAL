"# AI-Powered-Applications-Development---From-Zero-to-Hero-with-DIAL"

This repository contains exercises and example code for working with the DIAL API.

Structure:
- `ai-dial-chat-completions/` — chat client example using the DIAL proxy
- `ai-dial-models-parameters/` — parameter exploration examples

See per-folder README files for run instructions. Ensure you set the `DIAL_API_KEY` environment variable before running examples.
